# 🎮 NOVA — Gaming Platform

A futuristic **sci-fi gaming platform landing page** built using **HTML, CSS, and JavaScript**.
The project focuses on modern UI design, cinematic visuals, and smooth scroll-based animations to create an immersive gaming experience.

🔗 **Live Demo**
https://sakthisreya.github.io/NOVA---Gaming-Platform/

---

## ✨ Features

* 🎬 **Cinematic Hero Section** with dynamic video background switcher
* 🌌 **Smooth Scroll Animations** (blur, scale, fade effects)
* 📱 **Responsive Layout** optimized for desktop and mobile
* 🧩 **Interactive Feature Cards** with video previews
* 🎮 **Gamified Navigation System**

  * **SPAWN** – Welcome section
  * **LOADOUT** – Platform features
  * **BASE** – Contact section
* 🔗 Social media integration (GitHub & LinkedIn)

---

## 🛠️ Built With

* **HTML5**
* **CSS3**

  * CSS Grid
  * Flexbox
  * Scroll-driven animations
* **JavaScript**

  * Video switcher
  * UI interactions
* **Boxicons** (Icon library)
